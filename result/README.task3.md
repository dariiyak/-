# Задание 3: Поиск в глубину (DFS)

**Файл:** `task3.pl`

**Что делает**
- Предикат `dfs/3` находит путь от стартовой вершины к целевой в графе,
  заданном фактами `edge/2`, используя поиск в глубину.

**Интерфейс**
- `dfs(+Start, +Goal, -Path)`
  - `Start` — начальная вершина.
  - `Goal` — целевая вершина.
  - `Path` — список вершин от `Start` до `Goal`.

**Логика работы**
- DFS хранит список посещенных вершин, чтобы избежать циклов.
- Внутренний предикат `dfs/4` накапливает путь в обратном порядке.
- В конце путь разворачивается предикатом `reverse_list/2`.

**Требование к графу**
- Граф задается фактами вида `edge(U, V).`

**Пример использования**
```prolog
edge(a, b).
edge(b, c).
edge(a, d).
edge(d, c).

?- dfs(a, c, P).
P = [a,b,c] ;
P = [a,d,c].
```
