# Задание 2b: Реверсирование списка

**Файл:** `task2.b.pl`

**Что делает**
- Предикат `reverse_list/2` возвращает список в обратном порядке.

**Интерфейс**
- `reverse_list(+List, -Reversed)`
  - `List` — исходный список.
  - `Reversed` — список в обратном порядке.

**Логика работы**
- Используется хвостовая рекурсия с аккумулятором.
- Вспомогательный предикат `reverse_list/3` переносит элементы из исходного списка в аккумулятор.

**Пример использования**
```prolog
?- reverse_list([a,b,c], R).
R = [c,b,a].
```
