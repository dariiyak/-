# Задание 2a: Суммирование элементов списка

**Файл:** `task2.a.pl`

**Что делает**
- Предикат `sum_list/2` вычисляет сумму всех числовых элементов списка.

**Интерфейс**
- `sum_list(+List, -Sum)`
  - `List` — список чисел.
  - `Sum` — итоговая сумма.

**Логика работы**
- Базовый случай: сумма пустого списка равна `0`.
- Рекурсия: сумма `[H|T]` равна `H + SumTail`, где `SumTail` — сумма хвоста списка.

**Пример использования**
```prolog
?- sum_list([1,2,3,4], S).
S = 10.
```
